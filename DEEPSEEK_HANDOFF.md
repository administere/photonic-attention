# 光子 Attention 处理器 — DeepSeek 交接文档

## 1. 项目目标

验证一个可流片的**光子注意力点积计算核**的数值仿真。核心计算：用非负光强累积 + 双极差分探测计算 q·k，在所有物理非理想下保持 Spearman ρ > 0.99。

## 2. 架构概览

```
Q/K 向量 → 预畸变编码 → MZI sin² 传输 → 差分探测 → 数字还原点积

双极差分: (PP+NN) - (PN+NP)，天然对消共模噪声
预畸变:    inverse_pre_distortion(w) = 2·arcsin(√w)，抵消 sin² 非线性
```

## 3. 关键设计参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| delta_phi_max | π | 最大调制深度 |
| phi_static | 0 (A方案) 或 π/2 (A+C方案) | 静态偏置点 |
| optical_power | 2 mW | 总光功率 |
| dac_bits | 4 | DAC 量化位数 |
| calib_error | 2% | 校准误差 RMS |
| tau | 0 (瞬时) 或 1.0 (热滞后) | 调制器时间常数 |
| BANDWIDTH | 10 GHz | 探测器带宽 |

## 4. 已修正的关键错误

1. **预畸变**：使用真逆模型 `2·arcsin(√w)`，不是多项式拟合
2. **全局缩放**：Q/K 向量先统一缩放到 [-1,1]，再拆分正负部分
3. **噪声注入位置**：噪声施加在 `P_ideal = a_nonneg * power_per_ch * T` 之后，不是乘在 a 上
4. **偏置点认知**：A+C (phi_static=π/2) 会压缩透射范围到 [0.5,1]，反而可能降低保真度。A (phi_static=0) 使用全范围 [0,1]
